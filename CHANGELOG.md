# Changelog

All notable changes to `subscription-plans` will be documented in this file.

## [Unreleased]

### Added
- Initial package structure
- Plan and subscription management
- Feature-based usage tracking
- Trial and grace period support
- Multilingual support (Arabic and English)
- Event-driven architecture
- Comprehensive test suite

### Security
- Added `$fillable` properties to all models for mass assignment protection

## [1.0.0] - 2025-11-15

### Added
- First stable release
- Plan creation and management
- Subscription lifecycle management
- Feature usage tracking with automatic resets
- Multi-tenant support through polymorphic relationships
- Soft deletes for data history
- Observer pattern for subscription events

## Versioning

This package follows [Semantic Versioning](https://semver.org/):
- **MAJOR** version for incompatible API changes
- **MINOR** version for added functionality in a backwards compatible manner
- **PATCH** version for backwards compatible bug fixes
